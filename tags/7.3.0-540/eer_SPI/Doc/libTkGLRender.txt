Guide d'utilisation de la librairie "libTkGLRender-*.*.so"
-------------------------------------------------------

      Cette librairie permet d'utiliser OpenGL a l'interieur de Tcl/Tk. Elle gere plusieurs 
   chose dont:

      -L'initialisation d'OpenGL a l'interieur d'XWindows (GLX)
      -Le lien entre X, OpenGL et Tcl/Tk
      -La gestion des palettes selon les modes ColorIndex ou RGBA
      -La gestion des parametres de rendu d'OpenGL
      -La creation de postscript pour les primitives d'OpenGL par l'entremise du Feedback Buffer

      -et encore ...

      Cette librairie doit donc etre incluse ("load") des le depart d'un script Tcl/Tk pour bien 
   fonctionner. Des l'inclusion, un objet sera creer "GLRender" par lequel vous pourrez manipuler
   les divers parametres de la librairies
    

Parametres d'utilisations:
--------------------------

   load ../libTkGLRender-1.0.so GLRender

   GLRender 
 
      -aliasing     :Activation de l'anti-aliasing                          0   (0,1)
      -dithering    :Activation du degrade de tont (dithering)               0   (0,1)
      -filtering    :Activation du filtre des textures (bilinear-filtering)  0   (0,1)
      -shading      :Activation du (shading)                                 0   (0,1)
      -font         :Police de caractere a utiliser a l'interne              -   (Police au format X)
      -xmode        :Mode de rendu                                           CI  (CI,RGBA)
      -xbatch       :Mode batch                                              0   (0,1)
      -shutdown     :Fin du rendue et liberation de toutes les resources

      -stat         :Statistiques d'execution (retourne Le temps de rendu et le memoire residente)
      -info         :Information sur le renderer (GL_VENDOR, GL_VERSION ...) voir la documentation OpenGL
